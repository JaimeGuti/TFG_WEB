Visualización de Señal en Realidad Virtual

Trabajo Fin de Grado

Jaime Moraleda Gutiérrez

Resumen

Este Trabajo Fin de Grado consiste en la construcción de un sistema que permite la visualización de distintas señales y las operaciones básicas que se pueden aplicar sobre ellas mediante realidad virtual (VR) basado en tecnología A-Frame para la creación de escenas en 3D. El objetivo fundamental es hacer lo más sencillo posible la comprensión y aprendizaje de cada señal, así como los cambios que experimentan tras aplicarle una u otra operación, tanto gráficamente como matemáticamente.

This Thesis consists of the construction of a system that allows the visualization of different signals and the basic operations that can be applied to them through virtual reality (VR) based on A-Frame technology for the creation of 3D scenes. The main objective is to make it as easy as possible to understand and learn the graphic part of each signal, as well as the changes they undergo after applying one or another operation, both graphically and mathematically.

A-Frame

Es una tecnologı́a que nos proporciona un entorno de programación web enfocado al desarrollo y uso de experiencias de realidad virtual (VR).

Three.js

Es una librerı́a basada en JavaScript que aprovecha las capacidades de HTML5 y WebGL.

HTML5

El documento que lee el navegador para poder visualizar una web está escrito en el lenguaje de marcado HTML.

JavaScript

JavaScript es un lenguaje de programación utilizado principalmente para hacer páginas web interactivas en el lado del cliente.

DOM

Las siglas DOM significan Document Object Model, o lo que es lo mismo, la estructura del documento HTML.

WebGL

API multiplataforma que se utiliza para crear gráficos 3D acelerados por hardware en un navegador web sin necesidad de descargar ni instalar ningún complemento.

WebXR

Conjunto de estándares que se utilizan para la representación de escenas 3D en hardware.

Realidad Virtual (VR)

Es la creación y representación de un entorno de escenas y objetos simulados de apariencia real, generados informáticamente, que crea en el usuario la sensación de estar dentro de él.

Realidad Aumentada (AR)

Es el término que se usa para describir al conjunto de tecnologı́as que permiten que un usuario visualice una combinación entre el mundo fı́sico real a través de un dispositivo que le añade elementos digitales virtuales generados tecnológicamente.

Visual Studio Code

Editor de código fuente desarrollado por Microsoft y construido sobre el framework Electron.

GitHub

Es un servicio basado en la nube que aloja proyectos utilizando un sistema de control de versiones llamado Git.
